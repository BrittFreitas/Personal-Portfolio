MOODY MEMES

The Deets

Moody Memes is a React application that allows users to track their mood in gifs. Users can input their mood on a given day and choose a gif they think best represents it. Once selected, the user can store their Moody Meme in a timeline. This web app was designed and built in collaboration with Daniel Butcher, Kwame Appiah-Kubi, & Chris Taeyoung Kim using React, Firebase, and the GIPHY API.

The Stack

  • React
  • JavaScript
  • SASS
  • HTML5
  • CSS3
  • REST API
  • Firebase
